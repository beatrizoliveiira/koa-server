### Getting Started
- clone this repo
```sh
# install dependencies
yarn
# or
yarn install

# copy .env file
cp .env.example .env

# start project
yarn start

# see on GraphiQL graphql interface on localhost link
http://localhost:9000/graphql
```

### Generating the schema
```sh
yarn schema
```

### Create events manually
```sh
yarn seed
```
